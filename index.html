EvidenStamp 使い方

すべての処理はブラウザ内で完結。PDFは外部に送信されません。

基本的な使い方
  1. 設定パネルで符号・番号スタイル・区切り文字などを設定
  2. PDFをドロップゾーンにドラッグ＆ドロップ（またはクリックして選択）
  3. 必要に応じてファイルの並び替え・枝番設定
  4. 「スタンプを付与する」をクリック
  5. 各行の「DL」で個別保存、または「ZIP でまとめてDL」で一括保存
基本設定
符号甲・乙・丙・弁・資料・疎甲・疎乙から選択。「カスタム...」を選んで入力しEnterを押すと、プルダウンに追加されて次回以降ワンクリックで呼び出せます。複数被告がいる場合の「乙A」「乙B」や「別紙」なども登録可能。追加済みのカスタム符号を選ぶと「削除」ボタンが表示されます。
スタンプの形式PDF上に押されるスタンプの形式。正式（甲第1号証）・号証・シンプル・mints・ハイフンの5種類から選択。ファイル名の形式とは独立して設定できます。
ファイル名の形式リネーム後のファイル名に使われる形式。mints形式（甲001）が標準。スタンプとは独立して設定できます。枝番がある場合、「枝番の」は甲1の1、「枝番ハイフン」は甲1-1形式になります。
区切り文字ファイル名の証拠番号と書類名の間の区切り。mintsルールでは半角スペースが標準。
ファイル名変更チェックを外すとファイル名はそのままでスタンプのみ付与します。オフにするとファイル名の形式・区切り文字・元のファイル名を削除の設定がすべて無効になります。
元のファイル名を削除チェックをオンにすると「甲001.pdf」のように番号だけのファイル名になります。オンのときは区切り文字の設定が無効になります。
枝番の使い方

ファイル行右側の「枝」ボタンをオンにすると、そのファイルが枝番扱いになります。枝チェックを入れたファイル自身がその番号を持ち、「の1」から始まります。

メール本文.pdf　→　甲第2号証　（枝なし・通常）
添付PDF1.pdf　[枝]　→　甲第3号証の1　（枝チェックで自身の番号を取得）
添付PDF2.pdf　[枝]　→　甲第3号証の2
次の書証.pdf　→　甲第4号証

最初の枝ファイルの行に表示される「の 」入力で枝番の開始番号を変更できます（別の事件での続き番号から始めたい場合などに使用）。なお、1ファイル目でも枝番に設定できます。

その他の機能
並び替え各行左端の「☰」をドラッグして順序を変更できます。要素の上半分にドロップすると前に、下半分にドロップすると後ろに挿入されます。ファイルリスト右上の「番号順に整列」ボタンを押すと、ファイル名に含まれる数値（甲1_〇〇、甲2_△△など）を検出して自動的に並び替えます。並び替え後は再付与が必要です。
再付与「スタンプを付与する」はいつでも押し直し可能。設定変更後はそのまま押せば全ファイルを再処理します。
JPG・PNG画像ファイルをそのままドロップするとA4サイズのPDFに自動変換してスタンプを付与します。
白背景スタンプの背景を白くします。カラー書面や写真に重なっても読みやすくなります。
縦書きスタンプ縦書き文書用にスタンプテキストを縦組みで描画します。連続する数字は1スロットにまとめて表示されます（「12」が縦に並んでも崩れません）。「まとめて縦書き」で全ファイルに適用、各ファイル行の「縦」ボタンで個別設定も可能です。
正副本スタンプ各ファイル行のチェックをオンにすると正本・副本・控えのプルダウンが有効になります。正副本チェック済みファイルは証拠番号スタンプなしで左上に正副本スタンプのみ付与され、証拠番号の連番からも除外されます。
ページ番号付与各PDFの全ページ下部中央に「1 / 3」形式のページ番号を追加します。
枝番グループを結合ZIPダウンロード時に同じ号証番号のグループを1つのPDFにまとめます。
マイデフォルト「現在の設定をマイデフォルトに保存」で全設定を登録。「マイデフォルトに戻す」で呼び出せます。よく使う符号・スタイルを登録しておくと便利です。
メタデータ削除出力PDFから作成者・更新日時などのプロパティ情報を削除します。
設定の保存符号・スタイル・スタンプ設定などはブラウザに自動保存され、次回起動時に復元されます。
